# Infrastructure

Dossiers :
- `k3d/` — scripts de gestion du cluster k3d local
- `helm/` — values Helm (Ory, Postgres, Mailhog)
- `manifests/` — manifests K8s maison (Keto, ingress, jobs)
- `scripts/` — scripts d'initialisation et smoke tests
- `ci/` — workflows CI

Voir `docs/runbook/install-toolchain.md` pour l'installation de la toolchain.
